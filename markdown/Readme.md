## Code

### Inline code
You can print to the terminal using the `puts "hello world"` command.

### Multi line code

#### Without highlighting

```
def hello workd 
    puts"Hello world"
end
```

#### With highlighting

```rb
def hello workd     
    puts"Hello world"
end
```
